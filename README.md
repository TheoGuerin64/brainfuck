# Brainfuck
Optimized Brainfuck interpreter written in Rust!

## Usage
 - Clone the repo: `git clone https://github.com/TheoGuerin64/brainfuck.git`
 - Build: `cargo build -r`
 - Run: `cargo run -r -- assets/mandelbrot.bf`
